# Premium Plus Jobs

Welcome to the Premium Plus Jobs repository! This project serves as a dedicated platform for showcasing job openings at Premium Plus, a premier Zendesk partner in the EU. It aims to connect enthusiastic candidates with exciting career opportunities in customer experience consulting.

## Technical Overview

This repository is built using Jekyll, a static site generator, which allows for easy management and deployment of content. The site is designed to be user-friendly, providing clear navigation and an appealing layout for job seekers. It integrates Google Analytics for tracking site performance and uses various plugins to enhance functionality.

## Features and Functionality

- **Job Listings**: Displays open positions with detailed descriptions and application instructions.
- **404 Error Page**: Custom error handling to guide users back to the home page.
- **SEO Optimization**: Includes social meta tags and structured data to improve search engine visibility.
- **Responsive Design**: Ensures a seamless experience across devices with a mobile-friendly layout.
- **Google Analytics Integration**: Tracks visitor interactions for performance analysis.

## Important Files

- **_config.yml**: Contains configuration settings for the site, including the title, description, and Google Analytics ID.
- **index.md**: The homepage layout that introduces the site and lists job openings.
- **_layouts/**: Contains layout templates for different page types (home, page, post).
- **_posts/**: Directory for job postings, each represented as a markdown file with relevant metadata.
- **404.md**: Custom 404 error page to enhance user experience.

## Configurable Variables

The following variables can be adjusted in the `_config.yml` file:
- `title`: The title of the site.
- `description`: A brief description of the site.
- `url`: The base URL for the site.
- `google_analytics`: Google Analytics tracking ID.
- `author`: The name of the author or organization.

## Technologies and Dependencies

- **Jekyll**: Static site generator used for building the site.
- **Google Analytics**: For tracking and analyzing site traffic.
- **Liquid**: Templating language used in Jekyll for dynamic content rendering.
- **Sass**: CSS preprocessor for styling the site.

For further details, please refer to the individual files and documentation within the repository.
